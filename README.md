# Como executar o código HTML e CSS

## Após acessar este repositório no GitHub, basta fazer o download dos arquivos para o computador. Para isso, clique no botão “Code” e selecione “Download ZIP”. Em seguida, extraia o conteúdo do arquivo baixado e abra a pasta do projeto.

## Para executar, não é necessário instalar nada. Basta localizar o arquivo index.html (ou o arquivo principal do projeto) e abrir com qualquer navegador, como Google Chrome, Edge, Firefox ou Opera.
## O navegador carregará automaticamente o conteúdo do HTML e aplicará o estilo definido no arquivo CSS, desde que todos os arquivos estejam na mesma estrutura em que foram baixados.

## O projeto funciona de forma totalmente local, sem dependências externas.
